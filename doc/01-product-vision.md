# Quest Invite MVP - Product Vision

## One-line concept
Quest Invite is a playful, game-inspired invitation builder where users turn real-life plans into tiny RPG quests that can be shared through a link.

## Core idea
Instead of sending a plain message like:

> Want to grab ramen this Friday?

The user creates a mini quest card:

> The Great Ramen Expedition  
> Difficulty: Cozy  
> Reward: +50 Happiness  
> Accept Quest?

The recipient opens the link and experiences a short interactive scene with game-like visuals, animations, and an accept/decline response.

## Problem it solves
People often feel awkward asking others out, inviting friends, planning hangouts, or making a message feel special. Quest Invite makes invitations feel light-hearted, playful, and less intimidating.

## Target users
- Couples planning dates
- Friends planning hangouts
- Students inviting classmates
- Colleagues planning casual meetups
- Creators sending playful invites to their audience
- Event organizers who want simple fun RSVP pages

## Product positioning
Quest Invite is not a calendar app, not a dating app, and not a productivity tool.

It is:

> A playful invitation experience disguised as a tiny RPG quest.

## Emotional value
The product should feel:
- warm
- charming
- playful
- personal
- low-pressure
- memorable

## Key design principle
The app should solve real social friction using game-like presentation.

Users should think:

> This is cute. I want to send this to someone.

## MVP success criteria
The MVP is successful if a user can:
1. Create a quest invite in under 2 minutes.
2. Generate a shareable link without login.
3. Send it to someone.
4. Recipient opens it and instantly understands the invite.
5. Recipient feels entertained, not confused.

## Long-term vision
Quest Invite can eventually become part of a larger library of playful emotional utility apps, such as:
- Memory Capsules
- Potion Mail
- Decision Dungeon
- Cozy Campfire
- Confession Battle

All can share the same reusable interaction engine.
